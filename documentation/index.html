Google Map

To use Google Maps API you will need to get Google API Key and add it in HTML files at:

<script async defer src="https://maps.googleapis.com/maps/api/js?key=AIzaSyBToXJuBVmlW0oW-ZUkmFR_eN6s1QRK8tE&callback=initMap"></script>

To change the locate on the map, you will need to determine the coordinates in Google Maps by clicking at a certain point (as shown in the image below)

and then add the coordinates in the contact-us.html

<div class="gmap" data-lat="-33.878897" data-lng="151.103737"></div>

the full list of data attributes is

<div class="gmap" data-lat="-33.878897" data-lng="151.103737" data-center-lat="-33.876897" data-center-lng="151.106737" data-zoom="15" data-marker="./assets/images/parts/map-marker.png"></div>

Services & Projects categories

At Services and Projects list pages you can set categories filtration.

To asign some categories to service or project you should add data-groups data attribute to it

<div class="shuffle-item md-col-4 sm-col-6" data-groups='["Computer", "Laptop", "Smartphone"]'>...</div>

To add or change filtration links you should use data-filter data attribute

<a href="#" data-filter='["Laptop"]'>Laptop Repair</a>

Element Show Up Animation

To animate element when it appears in browser viewport you should add data-inview-showup data attribute to it with css classes for its animation

<div data-inview-showup="showup-translate-up">...></div>

As soon as you scroll to it classes from data-inview-showup data attribute will be added to element.

Predifined show up animation classes:

  • showup-translate-up - for move up animation
  • showup-translate-down - for move down animation
  • showup-translate-left - for move left animation
  • showup-translate-right - for move right animation
  • showup-scale - for scale up animation
  • showup-scale-down - for scale down animation
